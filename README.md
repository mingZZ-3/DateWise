## 💬 About
```
커플이 함께 데이트를 기록하며
가계부를 관리할 수 있는 서비스
```
* **월별 가계부** : 데이트 기록과 메모
* **유저 맞춤 데이트 장소 추천** : 최근 등록된 카테고리 기반 데이트 장소 추천
* **검색** : 날짜, 카테고리, 타입별로 지출/수입 내역 검색
* **분석** : 수입/지출, 카테고리별 지출, 월별 지출/수입 비교 분석


## 🛠️ Stack
* **Language** : JavaScript
* **Open API** : Kakao open api
* **Library & Framewrok** : Vue.Js, axios, mitt, pinia, vue-router
* **DB** : Json-server
